# 딸깍 콘테스트 문서 안내

## 버전 관리 규칙

- `Docs/` 아래 문서는 파일명 또는 문서 본문에서 버전을 식별할 수 있어야 한다.
- 협업용 문서의 권장 형식은 `name-vMAJOR.MINOR.md`다.
- 브랜치 이름의 `issueNum`은 GitHub 이슈 `#123`의 숫자 부분인 `123`을 의미한다.
- 프로젝트 문서는 한국어를 기본 언어로 사용한다.
