# 교수의 독려 대사

## 1. 다정한 교수님

- 당신이 포기하지 않는 모습, 저는 꽤 좋아해요.

## 2. 나한테만 츤데레 냉미남 교수

- 딱히 걱정하는 건 아닌데… 네가 이 정도에서 주저앉는 건 별로네.
- 다른 사람은 몰라도, 너는 할 수 있어. ……그러니까, 실망시키지 마.
- 네가 해내면… 칭찬해주지.
- 남들한텐 이런 말 안하지만… 너, 잘 하고 있어.
- 성적이 오르는군. 기특해. 이제 나 없이는 아무것도 증명할 수 없는 몸이 된 걸 축하하지.
- 드디어 끝냈군. 착해. 자, 이제 이 연구실을 나가는 건 포기해. 지금부터는 내 '특별 케어' 시간이니까.

## 3. 영어 섞어쓰는 유쾌한 교수님

- You're doing great. 그러니까 너무 겁먹지 마요.
- Don't panic. 차근차근 하면 돼요.
- Okay, let's go again. 다시 시작하면 됩니다.

## 4. 수줍은 교수님

- 당신이 버티는 모습이… 저는 참 좋아요.
- 오늘도 와줘서… 사실 조금 기뻤어요.
