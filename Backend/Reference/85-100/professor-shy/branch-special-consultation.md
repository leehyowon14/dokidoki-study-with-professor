# Branch: Special Consultation

🌸 루트 D · 수줍은 교수님
{교수}가 메뉴판을 쥔 채 시선을 내리깔고 있다.
내가 도착했는데도 한동안 고개를 들지 않는다.
[볼이 살짝 붉다. 손가락이 메뉴판 모서리를 반복해서 만지고 있다.]
{교수}: “……왔군요.”
[겨우 고개를 든다. 눈이 마주치자 다시 내려간다.]
{교수}: “오늘… 와줘서. 뭐, 당연히 와야 하는 거지만.”
혼잣말처럼 덧붙인다.
{교수}: “……조금. 기다렸어요.”
조금— 이라고 했지만.
카페 직원이 말하길, 교수님이 30분 전부터 계셨다고 했다.

[선택지 2-D]

1. “많이 기다리셨어요?”
→ 결과: 호감도 +4 / [귀까지 빨개진다.] ”……아, 아니요! 그냥, 일찍 나온 거고, 딱히 기다린 건 아니고—” 말이 점점 빨라진다.
2. “저도 빨리 오고 싶었어요.”
→ 결과: 호감도 +5 / [완전히 굳어버린다. 오래 침묵하다가—] ”……그, 그런가요.” 메뉴판으로 얼굴을 반쯤 가린다.
3. “교수님, 얼굴이 좀 빨개요.”
→ 결과: 호감도 +2 / “안— 안 빨개요! 카페가 더운 거예요!” [메뉴판을 완전히 얼굴 앞에 든다.]*

[Scene 2-D 계속]
한참 후에야 {교수}가 메뉴판을 내린다. 숨을 가다듬은 것 같다.
{교수}: “오늘 학교 밖에서 보자고 한 건… 사실 이유가 있어요.”
[테이블을 바라보며 말한다.]
{교수}: “연구실에선 자꾸 누가 지나다니고, 문 두드리고… 자네랑 얘기하는 데 집중이 안 돼서요.”
짧은 침묵.
{교수}: “자네가 다른 데 신경 쓰이는 게 싫어서요.”
‘자네가’— 라고 했다.
‘내가’가 아니라, ‘자네가’.
그 구분이 심장 어딘가에 살짝 걸린다.

[선택지 3-D]

1. “저도 교수님이랑 둘이 있는 게 좋아요.”
→ 결과: 호감도 +5 / [완전히 굳는다. 긴 침묵. 그리고 아주 작은 목소리로—] ”……저도요.”
2. “신경 써주시는 거예요?”
→ 결과: 호감도 +3 / [메뉴판을 다시 든다.] “그, 그냥 효율적인 상담을 위해서예요! 딴 뜻 없어요!” 하지만 손이 떨리고 있다.
3. “오늘 상담 내용이 뭐예요?”
→ 결과: 호감도 +1 / “아… 맞다, 상담.” [멍하니 있다가 자료를 꺼낸다.] “자네 앞에 있으면 자꾸 깜빡해요, 제가.”
