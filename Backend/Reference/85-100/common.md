# 85-100 공통 루트 · 특별 상담 챕터

📚 당신의 커리큘럼 — 시나리오 모듈
🌸 공통 루트 · 특별 상담 챕터

호감도 조건
• 이 챕터 진입 조건: 호감도 85~100
• 각 캐릭터 개별 루트 분기점

[Scene 1] 교외 상담 — 카페 도착

기말 시험 2주 전. {교수}에게서 메시지가 왔다.
“오늘 연구실 말고, 학교 밖으로 나오게. 학교 안에서는 집중이 안 되니까.”
평소와 다른 장소. 평소와 같은 {교수}. 그런데— 왜인지 오늘따라 달라 보인다.
카페 창가 자리. {교수}는 이미 와서 커피 두 잔을 앞에 두고 앉아 있다.
나를 발견하는 순간, 아주 짧게— 표정이 부드러워졌다가 이내 원래대로 돌아온다.
{교수}: “늦었군.”
시계를 본다. 약속 시간보다 3분 일찍 도착했다.
…늦지 않았는데. 그냥 그렇게 말하고 싶었던 것 같다.

[선택지 1]

1. “일찍 왔는데요, 교수님.”
→ 결과: 호감도 +2 / {교수} 잠깐 멈칫. ”……그렇군. 내가 더 일찍 온 거겠지.” 어색하게 커피를 밀어준다.
2. “죄송합니다, 바로 앉을게요.”
→ 결과: 호감도 +1 / {교수} 가볍게 고개를 끄덕인다. “됐어. 앉아.” 무난하게 시작.
3. (아무 말 없이 맞은편에 앉는다)
→ 결과: 호감도 +3 / {교수} 눈을 가늘게 뜨며 천천히 입꼬리가 오른다. ”…말 없이 바로 앉는군. 마음에 드는 반응이야.”

[Scene 3] 공통 · 카페를 나서며

상담이 끝났다. 아니— 끝났다고 하기엔, 오늘 나눈 대화가 ‘상담’이었는지 모르겠다.
{교수}와 나란히 카페 문을 나선다. 저녁 공기가 서늘하다.
{교수}가 잠깐 멈춰선다.
{교수}: “……오늘 어땠어?”
[나를 돌아보며. 표정은 각 루트마다 다르지만— 눈빛만큼은, 모두 같다.]
[기다리고 있다는 눈빛.]

[최종 선택지 — 전 루트 공통]

1. “좋았어요. 또 이런 날 있으면 좋겠어요.”
→ 결과: 호감도 +5 → 엔딩 분기: 해피 루트
2. “교수님이랑 있는 건 언제나 좋아요.”
→ 결과: 호감도 +4 → 엔딩 분기: 해피 루트
3. (아무 말 없이 웃는다)
→ 결과: 호감도 +3 → 엔딩 분기: 해피 루트
4. “그냥 상담이었잖아요.”
→ 결과: 호감도 -5 → 엔딩 분기: 노멀/배드 루트

🎬 엔딩 구조

💛 해피 엔딩 (호감도 88 이상 도달 시)
그날 이후로, {교수}가 달라졌다.
아니— 어쩌면 내가 달라진 건지도.
마지막 수업이 끝난 날, {교수}가 강의실 문을 닫고 조용히 말한다.
{교수}: “자네. 이번 학기로 내 수업은 끝이지.”
[잠시 멈추었다가.]
{교수}: “하지만 내 커리큘럼은 아직 끝나지 않았어.”
[천천히, 처음으로— 완전히 솔직한 눈빛으로 나를 본다.]
{교수}: “다음 학기도, 그다음도— 자네를 계속 볼 생각이야. 내가 설계한 커리큘럼엔 아직 챕터가 많이 남아 있거든.”
커리큘럼.
그게 공부를 말하는 건지, 다른 무언가를 말하는 건지—
이제는 구분하지 않기로 했다.
[HAPPY END — “당신의 커리큘럼은 계속됩니다”]

💙 노멀 엔딩 (호감도 70~87)
학기가 끝났다. {교수}와 나는 여전히 교수와 학생이다.
하지만 가끔— {교수}에게서 메시지가 온다.
“다음 학기에도 내 수업 들을 생각이야?”
그리고 마지막에 항상 한 줄이 더 붙는다.
{교수}: “……아니, 그것보다. 그냥 자네가 잘 지내는지 궁금해서.”
언젠가는, 우리가 어디쯤에 있는지 알 수 있을 것 같다.
[NORMAL END — “아직 펼치지 않은 챕터”]

🖤 배드 엔딩 (호감도 70 미만 또는 최종 선택지 4번)
학기가 끝났다. {교수}는 평소와 다르지 않다.
다르지 않은 척을 너무 잘한다.
마지막 날, 짧게 말한다.
{교수}: “수고했어. 좋은 학기였군.”
[뒤돌아서며.]
[표정은 보이지 않는다.]
뭔가를 놓쳤다는 느낌이 오래간다.
그게 무엇인지는— 다음 학기가 되어서야 알 것 같다.
[BAD END — “닫힌 챕터”]
